### Local Development

```bash
# Prerequisites: Ruby 3.x + Bundler

# Clone the repository
git clone https://github.com/Digidai/HireAI.git
cd HireAI

# Install dependencies
bundle install

# Run data validation
ruby _scripts/check_site_data.rb

# Start local server
bundle exec jekyll serve

# Visit http://localhost:4000
```
