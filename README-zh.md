## 为什么选择 HireAI？

## 研究与分析

### 旗舰报告

通过 Josh Bersin 的五阶段框架，对 46 个 HR AI 产品进行全面分析，研究从 1990 年代 ATS 系统到 2024+ 智能体 AI 平台的发展历程。

- 公司背景与历史

- 技术架构
- 目标市场与使用场景
- 优势与劣势
- 竞争定位
- 定价与部署方式

## 快速开始

### 在线浏览

访问 **[hireai.genedai.me](https://hireai.genedai.me)** 探索完整目录，支持搜索和筛选。

### 本地开发

```bash
# 前置条件：Ruby 3.x + Bundler

# 克隆仓库
git clone https://github.com/Digidai/HireAI.git
cd HireAI

# 安装依赖
bundle install

# 运行数据校验
ruby _scripts/check_site_data.rb

# 启动本地服务器
bundle exec jekyll serve

# 访问 http://localhost:4000
```

### 项目结构

```
HireAI/
├── _data/
│   ├── products.yml        # 产品数据库（149+ 产品）
│   └── tag_checklists.yml  # 评估标准（95 个标签）
├── _layouts/               # 页面模板
├── _scripts/               # 构建与校验脚本
├── tags/                   # 标签页（95 个）
├── *.md                    # 产品分析页（150+）
├── index.html              # 首页
└── product-directory.md    # 产品目录页
```
