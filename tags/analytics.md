## About HR Analytics

HR Analytics transforms recruiting data into actionable insights. From pipeline metrics to predictive modeling, analytics tools help organizations make data-driven talent decisions.

### Key Analytics Capabilities

| Capability | Value |
|------------|-------|
| **Pipeline Analytics** | Track candidates through each hiring stage |
| **Source Effectiveness** | Measure ROI by recruiting channel |
| **Time-to-Hire** | Identify bottlenecks in the process |
| **Quality of Hire** | Connect hiring data to performance outcomes |
| **Predictive Models** | Forecast hiring needs and success factors |

### Essential Recruiting Metrics

- **Time-to-Fill**: Days from requisition to offer acceptance
- **Cost-per-Hire**: Total recruiting spend divided by hires
- **Offer Acceptance Rate**: % of offers accepted
- **Source Quality**: Performance of hires by source
- **Diversity Metrics**: Representation across the funnel
