## About Candidate Sourcing

Sourcing is the proactive identification and engagement of potential candidates who may not be actively looking for jobs. Modern sourcing tools leverage AI, data aggregation, and automation to find the best talent faster.

### Sourcing Channels & Methods

| Channel | Description |
|---------|-------------|
| **LinkedIn** | Professional network with 900M+ profiles |
| **GitHub** | Developer talent based on code contributions |
| **Boolean Search** | Advanced search across multiple databases |
| **Social Media** | Twitter, Facebook, and niche communities |
| **Talent Rediscovery** | Mining your existing ATS database |

### Key Sourcing Metrics

- **Response Rate**: % of sourced candidates who reply
- **Qualified Rate**: % of responses that are job-fit
- **Time-to-Source**: Days to build a qualified pipeline
- **Cost-per-Hire**: Sourcing investment per successful hire
